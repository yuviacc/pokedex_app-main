# 📕 Pokédex App

A simple and interactive Pokédex web app built with Streamlit. Browse, search, and explore Pokémon data all in one place.

👉 **[Click here to try the app](https://pokedexappmain.streamlit.app/)**

---

## 🔍 Features

- Search for any Pokémon by name or number
- View stats, types, abilities, and more
- Clean and responsive UI powered by Streamlit
- Data sourced from the PokéAPI

## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```
